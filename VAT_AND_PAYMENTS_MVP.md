# MVP: НДС и заявки на оплату

## Целевой сегмент

Юрлица на ОСН с НДС и команды, где оплату должен согласовать финансовый директор. Для УСН или ИП без НДС достаточно проводок без налоговых полей.

## Что сделано в продукте

1. **Поля проводки:** `vat_rate` (ставка, %), `amount_excl_vat` (сумма без НДС). Полная номенклатура счетов-фактур и ЭДО в этой итерации не включалась — данные для управленческой аналитики и будущей выгрузки в бухгалтерию.
2. **Заявки на оплату:** сущность `payment_requests`, статусы `draft` → `submitted` → `approved` / `rejected` → `paid`. Утверждают роли `admin` и `finance`; новые пользователи по умолчанию `operator`, первый пользователь системы — `admin`.
3. **Журнал аудита:** `audit_log`, просмотр в профиле для `admin`.

## Рекомендации по развитию

- Выгрузка проводок в формате обмена с 1С (см. [INTEGRATIONS.md](./INTEGRATIONS.md)).
- Привязка утверждённой заявки к фактической транзакции одной кнопкой.
- Отдельный справочник контрагентов с ИНН и типом налогообложения.
