# ERP квадрат — описание приложения

Система управления коммерческой недвижимостью для владельцев нескольких объектов. Помогает отслеживать доходность, арендаторов, договоры и платежи.

---

## Технологии

| Слой | Стек |
|------|------|
| Frontend | React 18, Vite, TypeScript, React Router, Recharts, Axios |
| Backend | Node.js, Express, TypeScript |
| База данных | JSON (файловое хранение) |
| Авторизация | JWT, bcrypt |

---

## Что уже реализовано

### Авторизация
- Регистрация пользователя
- Вход по email/паролю
- JWT-токены
- Профиль пользователя (смена пароля, данные аккаунта)
- Тёмная/светлая тема

### Объекты недвижимости
- Список объектов
- Добавление, редактирование, удаление
- Детальная страница объекта (переход по клику)
- Поля: название, адрес, общая площадь

### Юниты (помещения)
- Список юнитов по объектам
- Добавление, редактирование, удаление
- Площадь, цена за м², статус (свободен/арендован/на ремонте)
- Авторасчёт месячной арендной платы

### Арендаторы
- Список арендаторов
- Добавление, редактирование, удаление
- Контакты: название, контактное лицо, телефон, email, ИНН

### Договоры аренды
- Список договоров
- Создание, редактирование, завершение
- Даты начала/окончания, ставка аренды, залог
- Связь юнит — арендатор
- Автоматическое обновление текущего арендатора в юните

### Платежи
- **Платежи от арендаторов:**
  - Генерация плановых платежей на месяц
  - Список платежей, отметка оплаты
  - Фильтр по месяцу, объекту, статусу
- **Обязательные платежи:**
  - Просмотр плановых обязательных платежей (коммунальные, налоги, обслуживание)
  - API для плановых и фактических обязательных платежей

### Финансы
- Страница «Финансы и платежи» с вкладками
- Список транзакций (API)
- Фильтры план/факт

### Дашборд (главная)
- Переключатель режима «План» / «Факт»
- Метрики: доходность, объекты, юниты, занятость, средняя ставка за м²
- Таблица объектов с данными

### Интерфейс и UX
- Боковая панель навигации (сворачиваемая, мобильная)
- Хлебные крошки
- Адаптивная вёрстка

## API (Backend)

Имеются endpoints для:
- `POST /api/auth/register`, `POST /api/auth/login`
- `GET/POST/PUT/DELETE /api/properties`
- `GET/POST/PUT/DELETE /api/units`
- `GET/POST/PUT/DELETE /api/tenants`
- `GET/POST/PUT/DELETE /api/leases`
- `GET/POST/PUT/DELETE /api/mandatory-payments`
- `GET/POST/PUT/DELETE /api/actual-mandatory-payments`
- `GET/POST/PUT /api/tenant-payments`
- `GET/POST /api/transactions`
- `GET /api/dashboard` — данные для главной страницы
