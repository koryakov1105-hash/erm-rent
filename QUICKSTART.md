# Быстрый старт

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Запустите проект:
```bash
npm run dev
```

Это запустит:
- Backend на http://localhost:3001
- Frontend на http://localhost:3000

## Первые шаги

1. Откройте браузер и перейдите на http://localhost:3000
2. Создайте первый объект недвижимости в разделе "Объекты"
3. Добавьте юниты к объекту в разделе "Юниты"
4. Система автоматически рассчитает месячную арендную плату на основе площади и цены за м²

## Структура данных

- **Объект** - здание или комплекс зданий
- **Юнит** - отдельное помещение в объекте
- При создании юнита система автоматически рассчитывает `monthly_rent = area × price_per_sqm`

## База данных

База данных SQLite создается автоматически при первом запуске в `backend/database/erm_rent.db`
